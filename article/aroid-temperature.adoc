= 耐えられる最低気温毎のサトイモ科まとめ

近縁種でも耐えれる温度には差があるので目安程度。

== 0度

ほとんどのサトイモ科植物は成長をやめてしまう。

* ミズバショウ
* ザゼンソウ
* アラム
* オロンチウム - 成長は止まる
* モンステラ - 旺盛ではなく耐えられる程度

== 10度

* フィロデンドロン - 葉が枯れ始める

== 15度

Colocasia、カラジウムなどが枯れ始める程度。
アグラオネマの体調が悪くなる程度。

* アグラオネマ - 葉が枯れる
* Colocasia（サトイモ）- 葉が枯れる
* カラジウム - 葉が枯れる

== 20度

* キルトスペルマ
* Lasia
